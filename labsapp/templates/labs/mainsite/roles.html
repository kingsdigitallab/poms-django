Roles in Transactions

The individuals and institutions in PoMS are often interconnected by participating in the same transactions. In the database, the appearance of an agent in a transaction if always linked to the specific 'role' the agent plays. Can we then discover any interesting pattern by examining these roles? Do agents tend to appear always in the same role, of are there exceptions to this rule?

Explore People's Roles in Transactions launch

This visualization tool allows you to compare the different roles two agents played in the context of all the transactions they have in common.
